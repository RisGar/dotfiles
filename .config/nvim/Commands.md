# Default Commands and Keybinds to remember

## Commands

- `:qa`: Quit all
- `:bw`: Close buffer

## Keybinds

- `C-o`: Go back after jump
- `za`: Toggle fold under cursor
- `z=`: Spelling suggestions
- `J`: Join lines when positioned at first line
- `C-z`: Suspend neovim, go back with `fg` command
